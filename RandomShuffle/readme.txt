Does a random shuffle preordering. The original order of nodes depends on many external factors. Doing a random shuffle tries to eliminate them.  

The software can be run as followed,
./RSH <filename> 

For example,
./RSH LiveJournal.txt (LiveJournal.txt is the input file)

Where the input file is a text file containing the graph in edge-list format. An example of the format of the input file is as followed,
0       1
0       2
1       3
